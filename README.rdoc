This is the code base used to power the zoe-doodle website on the server side. The purpose of this app is to collect money for the Juvenile Diabetes Research Fund
via selling home made ice cream and/or collecting donations. The site is also a teaching tool for handling transactions, integrating with other systems, and other fun stuff.

There are examples of Server Sent Events, transactional processing, delayed jobs, and more. Feel free to copy what you want.
